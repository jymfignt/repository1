# Learn Git Branching 学习笔记与实战

本项目记录了在 [Learn Git Branching](https://learngitbranching.js.org/) 网站的学习成果，以及日常开发中常用的 Git 命令速查表。

## 内容简介

项目主要包含以下两部分内容：

### 1. 习题通关答案
收录了 Learn Git Branching 网站各关卡的解题代码，涵盖：
*   **基础篇**：Commit, Branch, Merge
*   **进阶篇**：Rebase, Detached HEAD, Relative Refs (Tips: `^`, `~`)
*   **高级操作**：Cherry-pick, Interactive Rebase, Tags, Describe

### 2. Git 日常实战速查
整理了实际项目开发中的标准化流程：
*   **项目初始化**：`git init`, 关联远程仓库
*   **日常操作**：`add`, `commit`, `pull`, `push`
*   **分支管理**：新建、切换、合并及删除分支
*   **救火指南**：解决冲突、撤销修改 (`reset/revert`)、查看日志

## 快速开始

- 查看 **网站习题答案** 请参考项目内的对应文件。
- 查看 **日常命令速查** 请直接阅读笔记文档。


**Happy Gitting!**

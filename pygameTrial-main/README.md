# 🎮 2048物理合成游戏

一款创新的物理引擎驱动的数字合成游戏，融合了经典2048玩法与弹球物理机制。通过鼠标点击发射数字方块，利用物理碰撞和重力效果将相同数字的方块合并升级，达到目标分数即可通关！

## ✨ 特色功能

- 🎯 **物理发射系统** - 点击屏幕发射方块，真实的抛物线轨迹
- 🔄 **智能合成机制** - 相同数字自动合并升级（2→4→8→16→32→64）
- 💥 **三消爆破** - 三个64连成一线触发消除，获得高额分数
- 🎚️ **多难度选择** - 新手/初级/中级/高级四种难度
- 📚 **新手引导** - 分步教学帮助新玩家快速上手
- 🖼️ **自定义背景** - 支持上传PNG/JPG/BMP格式图片
- 🎆 **通关特效** - 精美的粒子烟花效果和庆祝音效
- 📊 **数据统计** - 实时显示分数、点击次数等数据

## 🎮 游戏玩法

### 基础操作
- **鼠标左键点击** - 向目标位置发射一串数字方块
- **A键** - 将所有方块向左移动
- **D键** - 将所有方块向右移动

### 游戏规则
1. 相同数字的相邻方块会自动合并并升级
2. 三个或更多64方块连成一线时触发消除
3. 达到目标分数即可通关
4. 合理利用物理碰撞和移动策略获得更高分数

### 难度设置
| 难度 | 目标分数 | 特点 |
|------|---------|------|
| 新手 | 500分 | 包含新手引导 |
| 初级 | 1000分 | 适合休闲玩家 |
| 中级 | 2000分 | 需要策略思考 |
| 高级 | 5000分 | 考验高级玩家 |

## 🚀 快速开始

### 环境要求
- Python 3.7+
- Pygame 2.x

### 安装依赖
```bash
pip install pygame
```

### 运行游戏
```bash
python game.py
```

### 文件结构
```
game_project/
├── game.py                    # 主程序
├── pic/                       # 图片资源
│   ├── background.png        # 默认背景图
│   ├── block.png             # 方块素材
│   ├── back.png              # 背景网格
│   └── blocknum/             # 数字方块图片
│       ├── 2.png
│       ├── 4.png
│       ├── 8.png
│       ├── 16.png
│       ├── 32.png
│       └── 64.png
└── snd/                       # 音效资源
    ├── 0.MP3                 # 合成音效
    ├── 1.MP3                 # 移动音效
    ├── 2.MP3                 # 发射音效
    ├── 3.MP3                 # 碰撞音效
    ├── 4.MP3                 # 消除音效
    └── congratulation.mp3    # 通关音效
```

## 📦 打包成EXE

### 安装打包工具
```bash
pip install pyinstaller
```

### 打包命令
```bash
pyinstaller --onefile --windowed ^
    --add-data "pic;pic" ^
    --add-data "snd;snd" ^
    --name "2048合成游戏" ^
    game.py
```

打包完成后，在 `dist` 文件夹中找到 `2048合成游戏.exe`

## 🛠️ 技术栈

### 核心技术
- **Python 3.x** - 开发语言
- **Pygame 2.x** - 游戏引擎框架
  - 窗口管理和渲染
  - 图像加载和处理
  - 音频播放系统
  - 事件处理系统

### 实现特性
- **自定义物理引擎** - 速度、加速度、碰撞检测
- **状态机管理** - 游戏流程和方块状态控制
- **粒子系统** - 烟花特效实现
- **智能算法** - 下落、合成、三消检测
- **资源管理** - 图片/音效预加载和缓存

### 算法亮点
- 九宫格邻近检测优化碰撞计算
- 从底部向上扫描的智能下落算法
- 横向+纵向双向三消检测
- 线性插值实现平滑移动动画
